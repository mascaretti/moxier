Rescaling the data

We rescale the data.

rescale <- function(x, na.rm = FALSE) {
  (x - min(x, na.rm = na.rm)) / (max(x, na.rm = na.rm) - min(x, na.rm = na.rm))
}
df_r <- as.data.frame(apply(X = df, MARGIN = 2, FUN = rescale, na.rm = FALSE))

and we plot it.
